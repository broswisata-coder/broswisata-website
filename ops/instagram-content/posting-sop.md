# Instagram Posting SOP

## Before Posting

Checklist:
- Use PNG for carousel uploads when available.
- Check that guest faces have permission.
- Remove any bank/payment information from visuals.
- Do not promise guaranteed wildlife sightings.
- Do not put fixed price unless Faris/Ahmad approve the current rate.
- Use CTA code in caption to track inquiry source.

## Caption Structure

Recommended order:
1. Strong first sentence.
2. 2 to 4 short paragraphs.
3. Who the route is best for.
4. CTA with keyword.
5. Hashtags.

Avoid:
- "Cheap package"
- "Guaranteed orangutan"
- "Best price"
- Too many mixed languages in one caption
- Overloaded hashtags

## Weekly Review

Every Sunday, record:
- Best reach post
- Best save post
- Best DM post
- Questions guests asked
- Which CTA code produced inquiry
- Photos needed from Ahmad next week

## Photo Shot List for Ahmad

Capture weekly:
- Ahmad greeting guests or checking route notes.
- Car road view without showing unsafe driving.
- Lodge exterior and breakfast setup.
- River/jungle trail details.
- Guest-approved group moment.
- Local food stop.
- Lake Toba/Samosir wide photo.
- Short 5-second vertical clips for Reels.
